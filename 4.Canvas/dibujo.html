Dibujo en canvas

¿Cuantas lineas quieres?


¿Que aprendí en este ejercicio?

* Crear un objeto en el DOM con la etiqueta 'canvas'
* Configurar las dimenciones del objeto (Alto y Ancho)
* Declarar el contexto del objeto (2d)
* Dibujar en el objeto con bloque de codigo desarrollado en un archivo.js (Dibujo creado a partir de lineas rectas)
* Crear un ciclo 'for' para que dibuje un cierto numero de lineas en diferentes posisicones y colores hasta formar una imagen
* Un cliclo puede ser creado dentro de una condicion que a su vez esta alojada en una funcion
* Crear cuadros de texto con una lista desplegable con distintas opciones para el usuario
* Crear un boton enviar y vicularle un evento al boton para que ejecute un bloque de codigo según la eleccion del usuario

Etiquetas y tecnicas aplicadas

HTLM

* Etiqueta: script + Atributo: src = (Usado para vicular un archivo.js a un archivo.html)
* Etiqueta: canvas + Atributos: width (ancho) y height (alto) = (Usado para crear un objeto canvas en el DOM y especificar sus dimenciones)
* Etiqueta: input + Atributos: type (tipo), value (valor) y id (identificador) = (Usado para crear cajar de texto, botones entre otros obtejos para la iteraccion del usuario)
* Etiqueta: article (articulo) = (Usada para representar un bloque de codigo contenido en un espacio especifico de la pagina web)
* Etiqueta: br (line break) = (Usada para producir un salto de linea en el texto)
* Etiqueta: h1 - h2 - h3 ... h6 (heading tags) = (Usados para crear el titulo principal 'h1' y los subtitulos 'h2, h3, etc' de la pagina web)
* Etiqueta: p (paragraph) = (Usada para crear parrafos)
* Etiqueta: select + Atributos: name (nombre) y id (identificador) + Etiqueta: option = (Usado para crear una caja de texto con despliegue de menú y definición de opciones)

Java Script

* Se utiliso un ciclo 'for' dentro de una condicion 'if' y 'else if' que a su vez esta dentro de una funcion creada
